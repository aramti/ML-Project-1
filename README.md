This project is a simple Machine Learning model to classify iris flowers into different species using the Iris dataset.

## Dataset
The dataset contains flower measurements:
   - Sepal Length
   - Sepal Width
   - Petal Length
   - Petal Width
     
Classes:
   - Setosa
   - Versicolor
   - Virginica

## Tech
- Python
- Pandas
- NumPy
- Scikit-learn
